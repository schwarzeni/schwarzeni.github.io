kiro 聊天记录中模型的输出内容人工导出为 markdown 格式

技术

kiro 还是挺好用的

最近在用 kiro 进行源码阅读，想把它在聊天框中输出的一些结论性的内容保存下来，但是如果直接复制粘贴的话，会丢失原有的格式，而当前 kiro 暂不支持聊天历史导出。在网上查了一些资料，摸索了一阵，找到了一个稍微有点麻烦的方法。

首先，打开开发者工具

然后选定需要导出的聊天内容，class 一般为 kiro-chat-message-markdown

然后将这个 html 复制出来

最后找一个 html 转 markdown 的工具，例如在线网站 https://htmlmarkdown.com/ ，转一下就行。网上 html 转 markdown 的工具还是挺多了，只要 html 的内容不是很复杂的话，基本都可以转成功。

最后希望 kiro 可以尽早支持聊天记录导出。
