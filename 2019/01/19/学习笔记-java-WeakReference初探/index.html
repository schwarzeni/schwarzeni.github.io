使用 WeakHashMap

它和 HashMap 的不同之处在于，它的 key 就是 WeakReference 类型，在调用其 put 方法是时如果是一个新key，那么就会使用这个 Entity 类继承 WeakReference 的类进行初始化，调用的是有两个参数的构造函数。

当这个 key 被 gc 回收时，对应的 value 也同时会被回收，这个键值对从map中被移除
