如何快速查看日历？

calendar-vim 可以满足你的需求，而且还可以把它变成日记本

  • :Calendar 垂直方向打开日历 (\cal)
  • :CalendarH 水平方向打开日历 (\caL)
  • :CalendarT 全屏显示当月日历

如果把它作为日记本的话需要在家目录下创建 ~/diary 文件夹

进阶用法详见文档 :help calendar
