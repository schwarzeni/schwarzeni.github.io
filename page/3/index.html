Golang I/O 的限流与计数

技术

这里的限流指，对于某个 I/O 接口，例如 HTTP handler 的 resp.Body，实现一个方法实现限流，比如限制这个接口的传输速度是 5MB/s 。而计数指，对于某个 I/O 接口，例如 io.Reader，实现一个方式实现对其 Read 方法读取数据速率的实时计算，这样子在实现 http get 的时候就可以实时打印下载速度了。
